# Relocation Evaluator

A personal research tool that evaluates US cities across ~65 fields — schools,
safety, climate, water/power infrastructure, land/legal, and
self-sufficiency — using web-search-grounded LLM research. Opening a GitHub
Issue with a city name triggers an automated fetch → render → deploy
pipeline; only the repo owner can trigger it.

**Browse evaluated cities:** https://mattmaximum.github.io/reloeval/
